本人将会尽可能保持该项目与原项目同步，如果有发现我偷懒了欢迎提Issue来催……

## 它能干啥？

DumbQQ可以：

+ 收发文字消息
+ 获取好友、群、讨论组、好友分组和最近会话的列表

DumbQQ不可以：

+ 包括但不限于收发图片、结构化消息、特殊消息
+ 上传/下载/发送文件
+ 代替你的QQ客户端
+ 当做女朋友替代品
+ 激活奥特曼变身器
